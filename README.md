# 获取上证交易所部分ETF历史数据

此仓库用于获取上证交易所部分[ETF](#ETF)的历史数据，时间从2020年1月1日至今。

## 环境

* Windows11

* Python 3.8.6

* claude

依赖包：

```bash
pip install requests plotly openpyxl
```

## 使用

```bash
python sse_etf_dashboard.py
```
